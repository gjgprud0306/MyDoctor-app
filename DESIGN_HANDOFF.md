## Assets

> 모든 시각 에셋은 Figma에서 직접 export한 파일만 사용합니다.
> Figma URL을 이미지 경로로 직접 사용하지 않습니다.
> 누락된 에셋은 대체 이미지를 만들지 않고 아래 누락 목록에 기록합니다.

| 항목 | 상태 | 저장 위치 |
|------|------|----------|
| 아이콘 export 완료 | 예 / 아니오 | hifi/public/assets/icons/ 또는 final/public/assets/icons/ |
| 일러스트 export 완료 | 예 / 아니오 | hifi/public/assets/images/illustrations/ 또는 final/public/assets/images/illustrations/ |
| 썸네일 export 완료 | 예 / 아니오 | hifi/public/assets/images/thumbnails/ 또는 final/public/assets/images/thumbnails/ |
| 배너 export 완료 | 예 / 아니오 | hifi/public/assets/images/banners/ 또는 final/public/assets/images/banners/ |
| 로고 export 완료 | 예 / 아니오 | hifi/public/assets/logos/ 또는 final/public/assets/logos/ |
| PWA 아이콘 export 완료 | 예 / 아니오 | hifi/public/assets/pwa/ 또는 final/public/assets/pwa/ |
